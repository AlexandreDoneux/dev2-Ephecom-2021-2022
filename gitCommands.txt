en ligne de commande:
git clone adresseCopiéDepuisLeSiteDeGit

git commit -m ="Le nom de la modification" => mise dans l'enveloppe
git push => envoyer vers le serveur
git pull => maj des données
git checkout developpement => passer sur la branche developpement
